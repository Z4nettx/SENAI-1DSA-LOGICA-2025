Exercícios de Lógica com Portugol

1 - Pedi um número inteiro ao cliente e com n1%==0 pude saber se é par ou impar e mostrar a ele.

2 - Pedi dois números reais e realizei a divisão pedida. Se um dos números for 0, o programa exibe uma mensagem de erro. Enquanto os números forem válidos, o programa pede números em loop para dividir.

3 - Pedi um número inteiro para o usuário, informei se o número está no intervalo de 10 e 50. O programa pede números até que o usuário digite um número fora do intervalo 10-50.

4 - Pedi um número inteiro positivo e realizei uma contagem regressiva a partir desse número até 0.

5 - Pedi que o usuário criasse uma senha e a confirmasse. Se a confirmação for diferente da senha criada, o programa pede novamente  até as senhas corresponderem.

6 - Com um número inteiro fornecido pelo usuário, apresentei a tabuada desse número de 1 até 10.

7 - Pedi uma letra ao usuário, se essa letra for vogal, o programa informa. Se for um caractere diferente, ou uma consoante, o programa informa. O programa repete a pedida até que o usuário digite um caractere que não seja uma letra .

8 - Pedi o peso e a altura de uma pessoa, com esses dados o programa calcula o imc da pessoa, informa, e classifica o imc dela e informa também.

9 - Pedi números em loop para o cliente. O programa só para quando o número fornecido pelo usuário for negativo, e quando acontecer, soma todos os números pares fornecidos pelo cliente.

10 - Pedi a idade do cliente, se a idade for >=18, mostrei que ele é maior de idade, senão, a idade é <18, logo, mostrei que ele é menor de idade.
